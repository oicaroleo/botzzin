🎯 PRÓXIMOS COMANDOS
═══════════════════════════════════════════════════════════════

1️⃣ PUSH PARA GITHUB (você executa no terminal):

git remote add origin https://github.com/SEU_USUARIO/botzzin.git
git branch -M main
git push -u origin main

ℹ️  Substituir SEU_USUARIO pela sua URL do repositório


2️⃣ EM RAILWAY (você faz na web):

- Criar conta em https://railway.app
- Conectar repositório GitHub
- Criar PostgreSQL
- Preencher variáveis de ambiente
- Deploy automático


3️⃣ TESTAR EM PRODUÇÃO (terminal):

curl https://seu-app-production.up.railway.app/health


4️⃣ COMPARTILHAR COM AMIGOS:

@seu_bot no Telegram
/start
Escolher plano
Testar PIX
