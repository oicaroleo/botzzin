# 🤖 Como Obter Token Telegram para BotZZIN

## Passo 1: Criar um Bot Telegram
1. Abra o Telegram
2. Busque por **@BotFather**
3. Clique em "Start"

## Passo 2: Criar Novo Bot
1. Digite: `/newbot`
2. Escolha um nome para o bot (ex: `MeuBot123`)
3. Escolha um username único (ex: `meubot123_bot`) - **DEVE terminar com `_bot`**

## Passo 3: Copiar Token
@BotFather vai responder com uma mensagem como:

```
Use this token to access the HTTP API:
1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghi
Keep your token secure and store it safely!
```

## Passo 4: Usar no BotZZIN
1. Volte ao BotZZIN (https://botzzin-production.up.railway.app)
2. Na dashboard, clique em **"+ Novo Bot"**
3. Cole o token completo (ex: `1234567890:ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghi`)
4. Clique em **"Criar Bot"**

## ⚠️ Importante
- **Não compartilhe seu token** - qualquer um com o token pode controlar seu bot!
- O token deve estar **completo** (com os dois pontos `:`)
- Pode levar alguns segundos para validar

## Próximos Passos após Criar
1. **Configurar**: Adicione mensagem de boas-vindas
2. **Planos**: Crie os planos de preço (Basic, Pro, etc)
3. **Métricas**: Veja os analytics quando tiver leads
